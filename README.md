# OxêMaps 🌎✨

## Objetivo
O OxêMaps é uma plataforma que conecta informações históricas e atuais de cidades do Nordeste, permitindo explorar locais e dados de forma interativa e tecnológica.

## Tecnologias
- Spring Boot  
- Thymeleaf  
- Bootstrap 5  
- Bootstrap Icons  
- Google Fonts (Fredoka One)  
- HTML, CSS, JavaScript
