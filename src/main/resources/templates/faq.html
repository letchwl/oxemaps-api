FAQ - OxêMaps

Conectando o Passado e o Presente utilizando a tecnologia
O que é o OxêMaps?
O OxêMaps é uma plataforma que conecta informações históricas e atuais, permitindo explorar locais e dados de forma interativa e tecnológica.
Como acessar diferentes cidades?
Você pode acessar diferentes cidades usando os botões na página inicial.
Posso contribuir com informações?
Sim! O OxêMaps permite que usuários cadastrados enviem dados e sugestões que passam por moderação antes de serem publicados.
Como entrar em contato?
Use a aba "Contato" na sidebar para enviar mensagens diretamente para a equipe do OxêMaps.
O OxêMaps é gratuito?
Sim, todos os recursos da plataforma são gratuitos para usuários e visitantes.
Funciona em dispositivos móveis?
Sim! O OxêMaps é totalmente responsivo, podendo ser usado em smartphones, tablets e desktops.
Como os dados são verificados?
Todos os dados enviados por usuários passam por uma moderação interna para garantir a veracidade das informações.
Como posso sugerir melhorias?
Você pode enviar sugestões através do formulário de contato ou via e-mail oficial da plataforma.
